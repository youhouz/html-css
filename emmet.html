HTML

Ajouter un élément enfant : ">"

Ajouter un frère

Multiplier un élément : "*"

Ajouter une CLASS : "."

Grouper les éléments multiples : "()"

Insérer du texte dans un élément : "{}"

Ajouter une numérotation : "$"

Commenter la numérotation à un chiffre donné : "$@X"

Numéroter dans l'ordre décroissant ( inverse ) : "$@-"

Remonter d'un élément enfant : "^"

Ajouter un attribut : [attribut="valeur"]

En bref

div.liste>ul>(li.objet$>p{Paragraphe}+img[src="http://www.20minutes"])+li.objet$@2*4>p{Paragraphe$@2}+img[src="img/bengal.jpg"]+a^^^footer>p{Et Voilà}

Le 1er groupe contient une li qui a 2 enfants : p qui contient du texte et une image qui a un attribut src.

Le groupe suivant contient les li de 2 à 5, en multipliant des lites dont les noms de class, contenu de paragraphre et valeurs attributs s"incrémentent

Les 3 ^^^ permettent de remonter les éléments parents pour placer le footer au même niveau que la div
